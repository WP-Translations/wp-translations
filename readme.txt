Import WordPress language packs from transifex.

== Description ==

WP-Translations is a free translation plugin that allows you to use your own translation files from any transifex.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-translations` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings -> WP-Translations screen to configure the plugin
